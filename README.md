akadvh
======

Akavh Vh Java Library und Console + GUI Client zum bequemen Abrufen der Noten

```
usage: java -jar akadvh.jar
 -c,--console       Consolenmodus
 -h,--help          Hilfe
 -m,--modul <arg>   Modul
 -p,--pass <arg>    Passwort
 -u,--user <arg>    Benutzername
 -v,--verbose       Mehr Ausgabe
 -version           Version
```
